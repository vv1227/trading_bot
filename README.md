\# 🚀 Binance Futures Trading Bot (Testnet)



\## 📌 Overview

A Python-based trading bot that allows users to place Market and Limit orders on Binance Futures Testnet using both CLI and a modern Streamlit UI.



\---



\## ✨ Features

\- ✅ Market \& Limit Orders

\- ✅ CLI Interface

\- ✅ Streamlit UI Dashboard

\- ✅ Logging \& Error Handling

\- ✅ Time Synchronization Fix

\- ✅ Testnet Trading (No real funds)
